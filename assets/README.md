# Assets Folder

Place your images and other media files here.

## Required Images:

1. **profile.jpg** - Your profile photo (recommended size: 300x300px or larger, square format)
2. **project1.jpg** - Screenshot/image for Project 1 (recommended size: 600x400px)
3. **project2.jpg** - Screenshot/image for Project 2 (recommended size: 600x400px)
4. **project3.jpg** - Screenshot/image for Project 3 (recommended size: 600x400px)

## Image Tips:

- Use JPG or PNG format
- Optimize images for web (keep file sizes reasonable)
- Use square images for profile photos
- Use landscape images for project screenshots
- You can use placeholder images from sites like:
  - https://placeholder.com
  - https://unsplash.com
  - https://picsum.photos

## Example placeholder URLs:

- Profile: https://via.placeholder.com/300
- Projects: https://via.placeholder.com/600x400
